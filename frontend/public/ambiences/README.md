# Ambiances (fichiers légers pour mobile)

Ce dossier est servi tel quel par Vite (URL: `/ambiences/<fichier>`).

## Recommandations d’optimisation mobile
- Format: **.mp3** ou **.ogg**
- Mono (souvent suffisant pour une ambiance)
- Sample rate: **22.05 kHz** ou **24 kHz**
- Bitrate: **64–96 kbps**
- Durée du loop: **20–60 secondes** (plus long = moins de répétition audible)

## Fichiers attendus (exemples)
- `forest.mp3` (forêt)

Tu peux remplacer / ajouter d’autres fichiers (pluie, vent…) si tu préfères du “réel” plutôt que du généré.
